# ☁️ Weather Dashboard

A live weather web application built with HTML, CSS, and Vanilla JavaScript.

## 🌐 Live Demo
🔗 [Click here to view the live app](https://vinaykosuri05.github.io/Weather-Dashboard)

## 📸 Preview
> Search any city and see live weather!

## ✨ Features
- 🔍 Search weather by any city in the world.
- 📍 Auto-detect current location using GPS.
- 📅 5-day weather forecast.
- 💧 Humidity, wind speed and weather condition display.
- ❌ Error handling for invalid cities and network issues.
- ⌨️ Press Enter key to search quickly.

## 🛠️ Built With
- HTML5 — Page structure
- CSS3 — Glassmorphism design, Flexbox, Animations
- Vanilla JavaScript — Logic, API calls, DOM manipulation
- [OpenWeatherMap API](https://openweathermap.org/api) — Live weather data
- [Unsplash](https://unsplash.com) — Dynamic weather backgrounds

## 📱 Works On
- ✅ Desktop browsers
- ✅ Mobile browsers
- ✅ Tablet

## 🚀 How to Run Locally
1. Clone this repository
```bash
git clone https://github.com/vinaykosuri05/Weather-Dashboard.git
```
2. Open the folder in VS Code
3. Run with Live Server extension
4. Replace API key in `script.js` with your own from [OpenWeatherMap](https://openweathermap.org/api)
